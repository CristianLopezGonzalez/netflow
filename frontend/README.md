# Frontend Netflow

SPA de Netflow construida con React + TypeScript + Tailwind.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Variables

Crea `frontend/.env` desde `frontend/.env.example`:

```env
VITE_API_URL=http://localhost:8000/api
```

## Flujo principal UI

- Login/registro con JWT
- Vista semanal de turnos
- Vista personal de tardes
- Alta y gestion de solicitudes de intercambio
- Visualizacion y compensacion de bolsa de dias
- Acciones de sincronizacion Google Calendar
